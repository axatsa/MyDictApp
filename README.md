# MyDict — Minimalist Language Learning App

A Babadum-inspired vocabulary app for English, Uzbek, and Korean.  
Live at: **mydict.classplay.uz**

---

## Stack

| Layer    | Technology |
|----------|------------|
| Backend  | Python 3.11 + FastAPI |
| Frontend | HTML5 + Tailwind CSS + Vanilla JS |
| Database | SQLite (shared global, via Docker volume) |
| AI       | Gemma 3 4B via Ollama (runs on host) |
| Proxy    | Traefik (uses existing `web` network) |

## Quick Start (Production)

### 1. Prerequisites on Server

```bash
# Docker + Docker Compose installed
# Traefik running on "web" external network
# Ollama installed on host

# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Pull the AI model
ollama pull gemma3:4b
```

### 2. Clone & Configure

```bash
git clone <your-repo-url> /home/temp/MyDictApp
cd /home/temp/MyDictApp
cp .env.example .env
nano .env   # Fill in your values (see comments in .env.example)
```

### 3. Deploy

```bash
chmod +x deploy.sh
./deploy.sh
```

That's it. The app will be live at `https://mydict.classplay.uz`.

## How It Works

1. **Word display** — App fetches a random word from SQLite. Background color changes on every tap.
2. **Desktop** — Large center word + translations. Example sentences scattered in screen corners.
3. **Mobile** — Word at top, scrollable sentence list at bottom.
4. **AI Generation** — Open Settings → paste your context → click "Update Word Base". Ollama (Gemma 3) generates 100 words in the background. Progress is polled every 5 seconds.
5. **Global DB** — All users share one SQLite database. Words generated by anyone are visible to everyone.

## DNS Setup (Cloudflare / your DNS provider)

Add an **A record**:
- Name: `mydict`
- Points to: `<your server IP>`
- Proxy: DNS only (or proxied)

Traefik handles SSL automatically via Let's Encrypt.

## Useful Commands

```bash

docker logs mydict_frontend -f

# Seed initial words manually
docker exec mydict_backend python seed.py

# Check word count in DB
docker exec mydict_backend python -c "from database import get_word_count; print(get_word_count())"

# Shell into backend
docker exec -it mydict_backend bash
```
